## Project Overview

**Horizon Courts** is a concept website for a professional tennis club offering coaching, court rentals, training programs, and events.

The project emphasizes:
- Clean layouts  
- Clear content hierarchy  
- Component-driven architecture for scalability  

It demonstrates how a real-world sports brand website can be structured using modern frontend tooling.

---

## Key Features

- Hero section with strong visual storytelling
- Modular React components
- Clean navigation with primary CTA (Book Now)
- Stats & facts section with visual indicators
- Services & training programs showcase
- Fully responsive layout
- Optimized asset handling via Vite

## Component Responsibilities

- **Navbar.jsx**  
  Handles top navigation, routing links, and the primary call-to-action.

- **Section1.jsx (Hero)**  
  Landing hero with headline, supporting text, and CTA.

- **Section2.jsx (About)**  
  Introduces Horizon Courts and its brand philosophy.

- **Section3.jsx (Facilities)**  
  Highlights courts, coaching quality, and amenities.

- **Section4.jsx (Stats & Numbers)**  
  Displays retention rate, active members, tournaments, and annual play hours.

- **Section5.jsx (Services)**  
  Training programs, private/group lessons, and court access.

- **Graph.jsx**  
  Visual representation of coach levels or player distribution.

- **Text.jsx**  
  Reusable text component for layout consistency.

---

## Design Decisions

- **Component-Based Layout**  
  Each section is isolated for maintainability and reuse.

- **Minimal Color Palette**  
  Conveys trust, professionalism, and a sports-first identity.

- **Strong Typography & Spacing**  
  Enhances readability and delivers a premium look.

- **Image-Driven UI**  
  Sports imagery builds realism and emotional engagement.
